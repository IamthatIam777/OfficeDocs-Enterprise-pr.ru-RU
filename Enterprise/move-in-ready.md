# <a name="move-in-ready"></a>Решения, не требующие сборки

 **Сводка.** Узнайте о службах облачного хранения, встроенных в облачные предложения корпорации Майкрософт.
  
Решения для хранения, не требующие сборки:
  
- используют готовые решения;
    
- готовы к использованию в комплекте с существующими службами;
    
- имеют ограниченные возможности настройки.
    
В следующих разделах подробно описаны все службы хранения, не требующие сборки.
  
## <a name="sharepoint-online-in-office-365"></a>SharePoint Online в Office 365

### <a name="features"></a>Возможности

- Файлы защищены, надежны и доступны с любого устройства
    
- Содержимое легко найти с помощью поиска
    
- Рабочие процессы и политики хранения упрощают использование и хранение содержимого
    
- Содержимое доступно через веб-API, браузер и в автономном режиме
    
### <a name="common-uses"></a>Использование

- Корпоративный портал или портал организации
    
- Совместная работа в группах
    
- Поиск в корпоративной среде
    
### <a name="key-storage-scenarios"></a>Основные сценарии хранения

- Совместная работа
    
- Внутренние коммуникации
    
- Управление видеоконтентом
    
### <a name="resources"></a>Ресурсы

Дополнительные сведения см. [здесь](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498).
  
Сведения о стоимости см. [здесь](https://products.office.com/business/compare-office-365-for-business-plans).

### <a name="common-uses"></a>Использование

## <a name="onedrive-for-business"></a>OneDrive для бизнеса

### <a name="features"></a>Возможности

- Хранилище защищено, надежно и доступно с любого устройства
    
- Пользователь может легко найти содержимое с помощью поиска
    
- Содержимое доступно через веб-API, браузер и в автономном режиме
    
- Каждый пользователь получает 1 ТБ места для хранения
    
### <a name="common-uses"></a>Стандартные варианты использования

- Хранилище документов для отдельных пользователей
    
- Нерегламентированная совместная работа над документами
    
### <a name="key-storage-scenarios"></a>Основные сценарии хранения

- Резервное копирование файлов
    
### <a name="resources"></a>Ресурсы

Дополнительные сведения см. [здесь](https://support.office.com/article/What-is-OneDrive-for-Business-187f90af-056f-47c0-9656-cc0ddca7fdc2).
  
Сведения о стоимости см. [здесь](https://products.office.com/business/compare-office-365-for-business-plans).

## <a name="microsoft-intune"></a>Microsoft Intune

### <a name="features"></a>Возможности

- Все развертываемое вами программное обеспечение упаковывается и отправляется в облачное хранилище Microsoft Intune.
    
- Подписка включает 20 ГБ места для хранения с возможностью приобретения дополнительного пространства, различные отчеты для программного обеспечения и оборудования, управление политиками, публикацию мобильных приложений, уведомления, мониторинг и соглашение об уровне обслуживания, гарантирующее показатель бесперебойной работы 99,99 %.
    
### <a name="common-uses"></a>Стандартные варианты использования

- Развертывание программного обеспечения
    
### <a name="key-storage-scenarios"></a>Основные сценарии хранения

## <a name="azure-site-recovery"></a>Azure Site Recovery

### <a name="features"></a>Возможности

- Простая автоматическая защита, репликация и восстановление в Azure
    
- Непрерывный мониторинг работоспособности и организованное восстановление
    
- Поддержка управляемых дисков и виртуальных машин Ubuntu
    
- Поддержка Windows Server 2016 и дисковых пространств
    
### <a name="common-uses"></a>Использование

- Виртуальные машины, работающие в среде с использованием Hyper-V, можно реплицировать между двумя центрами обработки данных или в Azure
    
- Решение InMage Scout позволяет защитить VMware и физические среды
    
### <a name="key-storage-scenarios"></a>Основные сценарии хранения

- Резервное копирование файлов
    
### <a name="resources"></a>Ресурсы

## <a name="azure-backup"></a>Azure Backup

### <a name="features"></a>Возможности

- "Прозрачная" защита данных, получаемых с Windows Server и Windows Server Essentials
    
- В состав решения входит System Center Data Protection Manager.
    
- Данные хранятся в геореплицированном хранилище (шесть копий ваших данных хранятся в двух центрах обработки данных Azure)
    
- Поддержка добавочных изменений, заданной частоты, сжатия, шифрования, увеличенных сроков хранения и регулирования полосы пропускания
    
### <a name="common-uses"></a>Использование

- Защита данных из различных локальных (SharePoint, SQL, Exchange) источников и источников в Azure (виртуальных машин)
    
### <a name="key-storage-scenarios"></a>Основные сценарии хранения

- Резервное копирование файлов
    
### <a name="resources"></a>Ресурсы

Дополнительные сведения см. [здесь](http://azure.microsoft.com/services/backup/).
  
Сведения о стоимости см. [здесь](http://azure.microsoft.com/pricing/details/backup/).
