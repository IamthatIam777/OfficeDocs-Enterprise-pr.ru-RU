# <a name="advanced-ediscovery-for-your-office-365-devtest-environment"></a>Advanced eDiscovery для среды разработки и тестирования Office 365

 **Сводка.** Настройка и демонстрация Office 365 Advanced eDiscovery с образцами данных в среде разработки и тестирования Office 365.
  
Расширенные eDiscovery Office 365 позволяет быстро найти и анализировать информацию по данным, которые хранятся в Office 365, включая электронной почты и документы. Это можно сохранить огромные объемы время и деньги, особенно в ситуациях, судебного разбирательства. Дополнительные сведения см в [Office 365 расширенного обнаружения электронных данных](https://support.office.com/article/Office-365-Advanced-eDiscovery-fd53438a-a760-45f6-9df4-861b50161ae4).
  
Выполнив действия, описанные в этой статье, вы создадите небольшой набор данных для урегулирования вымышленных разногласий по контракту и проанализируете его с помощью Advanced eDiscovery.
  
> [!TIP]
> Щелкните [здесь](http://aka.ms/catlgstack), чтобы просмотреть схему всех статей, относящихся к руководствам по лаборатории тестирования в One Microsoft Cloud.
  
## <a name="phase-1-create-your-office-365-devtest-environment"></a>Этап 1. Создание среды разработки и тестирования Office 365

Если необходимо проверить расширенной eDiscovery в облегченный путь с минимальным требованиям, следуйте инструкциям в этап 2 и 3 этапа из [Office 365 dev/тестовой среды](office-365-dev-test-environment.md).
  
Если вы хотите проверить расширенной eDiscovery в имитации enterprise, следуйте инструкциям в [DirSync для вашей среды разработки или тестирования Office 365](dirsync-for-your-office-365-dev-test-environment.md).
  
> [!NOTE]
> Тестирование расширенной обнаружения электронных данных не требуется имитации корпоративной среде, включающая имитации интрасети, подключенных к Интернету и синхронизации каталогов для леса Windows Server AD. Предоставляется здесь как вариант, чтобы тестирования и экспериментов можно выполнить в среде, которая представляет типичное организации. 
  
## <a name="phase-2-create-example-data-for-advanced-ediscovery"></a>Этап 2. Создание образца данных для Advanced eDiscovery

В этой процедуре мы создадим электронные сообщения, которые затем будут анализироваться в рамках дела Advanced eDiscovery.
  
1. Откройте Internet Explorer и войдите в систему в [https://outlook.com](https://outlook.com) для учетной записи Outlook, созданной на этапе 2[Office 365 dev/тестовой среды](office-365-dev-test-environment.md).
    
  - Если вы используете упрощенную среду разработки и тестирования, запустите частный сеанс Internet Explorer и войдите с локального компьютера.
    
  - При использовании среды разработки и тестирования имитации enterprise используйте Azure портала ([https://portal.azure.com](https://portal.azure.com)) для подключения на виртуальную машину CLIENT1 и затем вход из CLIENT1.
    
2. На вкладке **Почта Outlook** нажмите кнопку **Создать**.
    
3. В поле **для**, введите адрес электронной почты учетной записи User6 пробной подписки ( **user6 @.** <organization name> **. onmicrosoft.com**).
    
4. В поле темы введите **Тестовое сообщение 1**.
    
5. В тексте введите **Черновик контракта Tailspin**, а затем нажмите кнопку **Отправить**.
    
6. На вкладке **Почта Outlook** нажмите кнопку **Создать**.
    
7. В поле **Кому** введите электронный адрес учетной записи User6 в пробной подписке.
    
8. В поле темы введите **Тестовое сообщение 2**.
    
9. В тексте введите **Обеденное собрание Tailspin**, а затем нажмите кнопку **Отправить**.
    
10. На вкладке **Почта Outlook** нажмите кнопку **Создать**.
    
11. В поле **Кому** введите электронный адрес учетной записи User6 в пробной подписке.
    
12. В поле темы введите **Тестовое сообщение 3**.
    
13. В тексте введите **Разногласия по контракту Tailspin**, а затем нажмите кнопку **Отправить**.
    
14. Щелкните значок пользователя в правом верхнем углу и выберите команду **Выход**.
    
15. Откройте новую вкладку и войдите в портал Office 365 ([https://portal.office.com](https://portal.office.com)) с именем учетной записи и пароль для учетной записи User6 пробной подписки.
    
16. На вкладке **портала Office 365** выберите элемент **Почта**.
    
17. На вкладке **Outlook почты - User6 -** проверьте, что User6 полученных все три сообщения электронной почты из учетной записи электронной почты Outlook.
    
18. Переключитесь на **вкладку портала Office 365** для пользователя User6.
    
19. Нажмите значок пользователя в правом верхнем углу страницы, а затем нажмите **Выйти**.
    
В этой процедуре мы создадим два документа Word, которые затем будут анализироваться в рамках дела Advanced eDiscovery.
  
1. На странице **Office** нажмите кнопку **Войти** и войдите с помощью учетных данных глобального администратора.
    
2. На новой вкладке, доступ к URL-адрес сайта SharePoint производственной: **https://** <fictional organization name> **.sharepoint.com/sites/production**
    
3. На вкладке **Семейство рабочих веб-сайтов** найдите раздел **Документы** и выберите элементы **Создать > Документ Word**.
    
4. На странице **Word Online** введите **Черновик контракта Tailspin**, подождите, пока в заголовке не появится надпись **Сохранено**, а затем закройте вкладку **Word Online**.
    
5. На вкладке **Семейство рабочих веб-сайтов** найдите раздел **Документы** и выберите элементы **Создать > Документ Word**.
    
6. 	На вкладке **Word Online** введите **Примечания к разногласиям по контракту Tailspin**, подождите, пока в заголовке не появится надпись **Сохранено**, а затем закройте вкладку **Word Online**.
    
7. В списке документов на вкладке **Семейство рабочих веб-сайтов** должны отобразиться файлы **Документ** и **Документ1**.
    
8. Закройте вкладку **Семейство рабочих веб-сайтов**.
    
## <a name="phase-3-use-advanced-ediscovery-for-a-legal-dispute"></a>Этап 3: Использование расширенных обнаружения электронных данных для юридических споров по

К сожалению споров по контракт между вашей организацией и Tailspin Toys достиг точки юридических целях. В этой процедуре создайте и настройте вариантом eDiscovery расширенный поиск и анализ электронной почты и документы, содержащие текст «Tailspin контракт».
  
Порядок использования Advanced eDiscovery:
  
- Создать поиск в системы &amp; центре соответствия требованиям, анализ его результатов, а затем подготовьте результаты расширенной обнаружения электронных данных.
    
- Создайте и настройте дело в Advanced eDiscovery и проанализируйте результаты поиска.
    
В этой процедуре создается поиска в системы &amp; соответствия центра «Tailspin контракт» поиск в окне результатов, а затем подготовьте результаты расширенной обнаружения электронных данных.
  
1. На вкладке **портала Office 365** выберите элемент **Администрирование**.
    
2. В левой области навигации на вкладке Центра администрирования выберите элементы **Центры управления > Соответствие требованиям**.
    
3. На **безопасности &amp; соответствия** нажмите кнопку **разрешения**.
    
4. В списке **Разрешения** дважды щелкните **Управление организацией**.
    
5. В окне **Группы ролей** нажмите значок "плюс" в разделе **Члены**.
    
6. В окне **Выбор членов** дважды щелкните имя учетной записи администратора, а затем нажмите кнопку **ОК**.
    
7. В окне **Группа ролей** нажмите кнопку **Сохранить**.
    
8. В списке **Разрешения** дважды щелкните элемент **Руководитель службы обнаружения электронных данных**.
    
9. В окне **Группа ролей** нажмите значок "плюс" в разделе **Администратор, ответственный за обнаружение электронных данных**.
    
10. В окне **Выбор членов** дважды щелкните имя учетной записи администратора, а затем нажмите кнопку **ОК**.
    
11. В окне **Группа ролей** нажмите кнопку **Сохранить**.
    
12. На панели навигации слева щелкните **поиска &amp; поиска контента расследования >**.
    
13. Нажмите значок "плюс", чтобы добавить операцию поиска.
    
14. В окне **Новый поиск** введите **Поиск контракта Tailspin** в поле **Имя**.
    
15. В разделе **Где нужно искать?** нажмите **Поиск везде**, выберите **Exchange**, **SharePoint** и **Общие папки**, а затем нажмите кнопку **Далее**.
    
16. В разделе **Что вы ищете?** введите **Контракт Tailspin**, затем нажмите кнопку **Искать**.
    
17. В списке операций поиска выберите имя **Поиск контракта Tailspin**.
    
18. В области **поиска контракт Tailspin** щелкните **Просмотр результатов поиска** в области **результатов**. Вы должны увидеть окна со списком двух документов на сайте SharePoint производственной ( **документов** и **Документ1**) и сообщения **электронной почты Test 1** и **тестирования адрес электронной почты 3** для User6. Закройте окно.
    
19. В области **Поиск контента**, в разделе **Анализ результатов с помощью Advanced eDiscovery**, нажмите **Предварительный просмотр результатов для анализа**.
    
20. В окне **Подготовка результатов поиска "Поиск контракта Tailspin"** нажмите **Подготовить** и дождитесь завершения операции.
    
В этой процедуре мы создадим дело для Advanced eDiscovery и проанализируем результаты поиска контракта Tailspin.
  
1. На панели навигации слева щелкните **обнаружения электронных данных** в разделе **поиска &amp; расследования**.
    
2. В области **Обнаружение электронных данных** нажмите **Перейти к Advanced eDiscovery**.
    
3. На вкладке **Advanced eDiscovery** нажмите значок "плюс", чтобы добавить дело.
    
4. 	В области **Добавление дела** введите **Разногласия по контракту Tailspin** в поле **Имя**, а затем нажмите кнопку **ОК**. Дождитесь создания дела.
    
5. Дважды щелкните дело **Разногласия по контрактам Tailspin** в списке. 
    
6. В списке **контейнер** щелкните контейнер **Tailspin контракт поиска** и нажмите кнопку **процесс**. Ожидание обработки для выполнения.
    
7. Когда в нижней части окна появится надпись **Обработка: завершена**, выберите элемент **Сводка по обработке** в левой области навигации, чтобы открыть сводку.
    
8. В верхней области навигации нажмите **Анализировать**.
    
9. На странице **Настройки**, в разделе **Темы**, введите значение **3** в поле **Максимальное количество тем**.
    
10. Щелкните **Анализ** и дождитесь выполнения анализа. Вы должны увидеть ряд круговой диаграммы с помощью сведений из целевого заполнения, документы, сообщения электронной почты и вложения. Для получения дополнительных сведений см. [Просмотр анализ результатов](https://support.office.com/article/Viewing-Analyze-results-5974f3c2-89fe-4c5f-ac7b-57f214437f7e).
    
Теперь вы можете использовать эту среду для создания контента, операций поиска и дел, а также дальнейших экспериментов с Advanced eDiscovery в Office 365.
