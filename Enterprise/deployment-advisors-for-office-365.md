# <a name="deployment-advisors-for-office-365-services"></a>Помощники по развертыванию служб Office 365

Рекомендации по развертыванию Office 365 предоставляют вам самостоятельную поддержку по настройке Office 365 для вашей организации. Просто выберите руководство или помощник по настройке и войдите в Office 365. 

При выборе компонентов и параметров, которые необходимо развернуть, помощник создает пошаговые планы установки, которые настроены в соответствии с вашими потребностями. Вы получите полный набор инструкций, видеороликов, справочных статей и сценариев. Некоторые рекомендации имеют автоматизацию, которая изменит некоторые параметры, а в других областях помощника вы измените свои параметры и данные с помощью помощника. Эти рекомендации можно использовать в любое время, даже при планировании или после того, как вы уже настроили службы, чтобы узнать больше о возможностях и вариантах Office 365.
  
## <a name="windows-10-with-office-365"></a>Windows 10 с Office 365

[Помощник по развертыванию Microsoft 365](https://aka.ms/microsoft365setupguide)
  
Microsoft 365 это комплексное интеллектуальное решение, включающее в себя Office 365, Windows 10 и Enterprise Mobility + Security. Microsoft 365 позволяет всем пользователям творческо работать и работать вместе. Используйте этот помощник для настройки устройств с Windows 10 или обновления компьютеров пользователей до Windows 10, с необязательным развертыванием приложений Office, Windows Analytics и Advanced Threat Protection в Защитнике Windows (только для Microsoft 365 Enterprise с планом а).


## <a name="mail-migration-and-protection"></a>Миграция и защита почты

### <a name="prepare-your-environment"></a>Подготовка среды 
[Руководство по подготовке среды](https://go.microsoft.com/fwlink/?linkid=2005213) — это место для запуска. Расскажите нам о задачах развертывания, чтобы убедиться, что вы добавляете домены, создаете пользователей и назначаете лицензии в правильном порядке. Это особенно важно, если вы планируете перенести электронную почту или настроить гибридное развертывание. 

### <a name="exchange-migration-advisor"></a>Помощник по миграции Exchange
[Помощник по миграции почты Office 365](https://aka.ms/office365setup) поможет вам переместить почтовые ящики из текущей почтовой системы в Exchange Online в Office 365 с помощью автоматизированных средств и пошаговых руководств. Мы рекомендуем оптимальный путь миграции для вашей организации на основе текущей почтовой системы, количества почтовых ящиков, которые вы хотите перенести, и планируемого управления доступом пользователей и пользователей.
  
В разделе " [Контакты и Календарь Gmail](https://aka.ms/gmailcontactscalendar) " представлены пошаговые инструкции по переносу контактов Gmail и элементов календаря Google в Office 365. При переносе почтового ящика Gmail пользователя в Office 365 сообщения электронной почты переносятся, но контакты и элементы календаря не удаляются. В этом руководстве представлены действия для импорта контактов Gmail и элементов календаря Google в Office 365.
  
### <a name="exchange-online-protection"></a>Exchange Online Protection
Microsoft [Exchange Online Protection (EOP)](https://aka.ms/EOPguidance) — это облачная служба фильтрации электронной почты, которая помогает защитить организацию от нежелательной почты и вредоносных программ и включает функции для защиты Организации от нарушений политики обмена сообщениями.

Включены инструкции по использованию System Center Configuration Manager, сценария запуска групповой политики, образа диска операционной системы или общей активации служб удаленных рабочих столов (RDS).

### <a name="sharepoint-online"></a>SharePoint Online
  
На панели [быстрого запуска SharePoint Online](https://aka.ms/SPOquickstartguide) показано, как настроить SharePoint в облаке для хранилища документов и управления контентом. Эта базовая настройка необходима, если вы не используете большой объем данных для переноса или локальный сервер SharePoint, который вы хотите продолжить в гибридной конфигурации.
  
[Помощник по развертыванию SharePoint Online](https://aka.ms/spoguidance) помогает выбрать вариант развертывания, который лучше всего подходит для вашей организации, и предоставляет инструкции по настройке компонентов SharePoint Online в соответствии с бизнес-потребностями. Выбор между облачным развертыванием, гибридным развертыванием или локальной миграцией в облако. Затем выполните действия для настройки функций SharePoint Online, таких как хранение файлов и общий доступ, общий доступ к файлам, семейства веб-сайтов, глобальные параметры и профили пользователей, а также сайт группы Office 365.
  
## <a name="security-and-identity"></a>Безопасность и удостоверение

### <a name="azure-active-directory-connect-azure-ad-connect-advisor"></a>Советник по подключению Azure Active Directory (Azure AD Connect)
[Помощник по Azure AD Connect](https://aka.ms/aadconnectpwsync) — это пошаговое руководство по добавлению учетных записей пользователей в Office 365 без ручного создания каждого пользователя. Здесь также показано, как настроить синхронизацию хэша паролей или сквозную проверку подлинности, чтобы пользователи могли войти в свою электронную почту и ваш домен, используя один и тот же пароль. Если вы решили настроить федеративный вход с помощью AD FS, в этом руководстве также приводятся инструкции по развертыванию AD FS на новом сервере или в существующей ферме Windows Server 2012 R2. Azure AD Connect заменяет более ранние версии средств интеграции удостоверений, таких как DirSync и Azure AD Sync, и в основном используется для добавления пользователей и других данных в Azure Active Directory для Office 365.
  
### <a name="azure"></a>Azure
[Руководство по установке Azure AD](https://aka.ms/azureadbasic) позволяет настраивать функции, такие как управление доступом на основе групп, сброс паролей для облачных приложений и прокси-сервер приложения Azure Active Directory для публикации локальных веб-приложений.
  
[Руководство по установке Azure AD Premium](https://aka.ms/aadpguidance) позволяет включить многие функции Azure Active Directory Premium (Azure AD Premium), которые предоставляют возможности управления удостоверениями, такие как многофакторная проверка подлинности, единый вход (SSO), регистрация устройств самостоятельное управление паролями и группами и наблюдение за безопасностью.
  
### <a name="verify-your-domain"></a>Verify your domain
[Проверка домена в office 365](https://aka.ms/verifyyourdomaino365) Advisor поможет вам настроить Office 365, добавив собственное доменное имя (также называемое именем веб-сайта).
  
## <a name="communication-and-online-conferencing"></a>Связь и конференц-связь

### <a name="office-365-groups"></a>Группы Office 365
[Руководство по настройке групп Office 365](https://aka.ms/groupsguide)

Группы Office 365 — это общая рабочая область для сообщений электронной почты, бесед, файлов и событий, в которых участники групп могут собирать все дела. Группы в Office 365 позволяют выбрать набор людей, с которыми вы хотите сотрудничать, и легко настроить набор ресурсов для совместного использования. Вам не нужно будет вручную назначать разрешения всем этим ресурсам, так как при добавлении участников в группу им будут предоставлены разрешения, необходимые для предоставления группе инструментов.
  
### <a name="microsoft-teams"></a>Microsoft Teams

[Помощник по развертыванию Teams](https://aka.ms/teamsguidance)
  
Microsoft Teams — это служба совместной работы на основе чата в Office 365, предоставляющая рабочие области групп для чатов, звонков, собраний и конфиденциальных сообщений. Помощник по развертыванию Teams помогает настроить и настроить оптимальный интерфейс Teams с помощью групп Exchange Online, SharePoint Online, OneDrive для бизнеса и Office 365. Он включает сведения о настройке параметров клиента, Teams и каналах для проектов, звонков и собраний, сообщений, вкладок для быстрого доступа к приложениям, информационным Боты и соединителям сторонним службам.

[Помощник по развертыванию Skype для бизнеса](https://aka.ms/skypeguidance)
  
Помощник по развертыванию Skype для бизнеса поможет вам начать работу со Skype для бизнеса Online. В этом помощнике показано, как настроить функции Skype для бизнеса, например обмен мгновенными сообщениями, собрания по сети, видеоконференции, Конференц-связь с телефонным подключением по телефонной линии (PSTN), УАТС и трансляцию собраний Skype. В этой статье приведены инструкции по настройке гибридной среды, в которой вы подключаете Skype для бизнеса Online к локальному развертыванию Skype для бизнеса.
