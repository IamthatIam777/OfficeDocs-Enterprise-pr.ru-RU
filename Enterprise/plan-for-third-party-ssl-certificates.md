# <a name="plan-for-third-party-ssl-certificates-for-office-365"></a>Планирование сторонних SSL-сертификатов для Office 365

 **Сводка:** Описываются SSL-сертификаты, необходимые для Exchange локального и гибридного, единый вход с использованием служб AD FS службы Exchange Online и веб-служб Exchange. 
  
Чтобы зашифровать обмен информацией между клиентами и средой Office 365, сертификаты сторонних производителей Layer сокетов (SSL) должны устанавливаться на серверах инфраструктуры.

||
|:-----|
| В этой статье является частью [сети планирование и настройка производительности для Office 365](https://aka.ms/tune).|
   
Сертификаты требуются для следующих компонентов Office 365:
  
- локальная среда Exchange;
    
- Единый вход (SSO) (как для серверов федерации Active Directory (AD FS), так и для прокси-серверов федерации AD FS)
    
- Exchange Online services, такие как автообнаружение, мобильный Outlook и веб-служб Exchange
    
- гибридный сервер Exchange.
    
## <a name="certificates-for-exchange-on-premises"></a>Сертификаты для локальной среды Exchange

Общие сведения об использовании цифровых сертификатов для защиты связи между локальной организации Exchange и Exchange Online, в статье TechNet [Требования к сертификатам общее представление о](https://go.microsoft.com/fwlink/p/?LinkID=243657).

## <a name="certificates-for-autodiscover-outlook-anywhere-and-active-directory-synchronization"></a>Сертификаты для автообнаружения, мобильный Outlook и синхронизации Active Directory

Внешнем Exchange 2013, Exchange 2010, Exchange 2007 и клиентского доступа Exchange 2003 серверов (классов) требуется стороннего сертификата SSL для безопасных подключений для синхронизации службы автообнаружения, мобильный Outlook и Active Directory. Возможно, уже установлен в локальной среде сертификата.
  
## <a name="certificate-for-an-exchange-hybrid-server"></a>Сертификат для гибридного сервера Exchange

На внешнем гибридный сервер Exchange или серверов требуется стороннего сертификата SSL для безопасного подключения к службе Exchange Online. Необходимо получить сертификат стороннего поставщика SSL.
