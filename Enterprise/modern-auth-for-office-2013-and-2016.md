# <a name="how-modern-authentication-works-for-office-2013-and-office-2016-client-apps"></a>Современная проверка подлинности для клиентских приложений Office 2013 и Office 2016

Ознакомьтесь с этой статьей, которая Узнайте, как использовать функции современных проверки подлинности в Office 2013 и Office 2016 клиентских приложений на основе проверки подлинности конфигурации на клиента Office 365 для Exchange Online, SharePoint Online и Скайп для бизнеса в Интернет.
  
## <a name="availability-of-modern-authentication-for-office-365-services"></a>Доступность современных проверки подлинности для служб Office 365

Для служб Office 365 — это состояние по умолчанию современных проверки подлинности:
  
- **Включается для Exchange Online по умолчанию.** Просмотрите [включения или отключения проверки подлинности на современном в Exchange Online](https://support.office.com/article/58018196-f918-49cd-8238-56f57f38d662) в его включить или отключить. 
    
- **Включается для SharePoint Online по умолчанию.** 
    
- **Включается для Скайп для бизнеса в Интернет по умолчанию.** В разделе [Включение Скайп для бизнеса в Интернет для современных проверки подлинности ](https://social.technet.microsoft.com/wiki/contents/articles/34339.skype-for-business-online-enable-your-tenant-for-modern-authentication.aspx)в его включить или отключить.
    
## <a name="sign-in-behavior-of-office-client-apps"></a>Поведение входа в клиентских приложениях Office

Клиентские приложения Office 2013 поддерживает проверки подлинности по умолчанию. Прежних версий означает, что они поддерживают либо входа в Microsoft Online помощника или базовой проверки подлинности. Чтобы для этих клиентов для использования возможностей современных проверки подлинности Windows, клиент имеет иметь разделы реестра, настроенные. Сведения содержатся в разделе [Включение современных проверки подлинности для Office 2013 на устройствах Windows](https://support.office.com/article/7dc1c01a-090f-4971-9677-f1b192d6c910).
  
Ознакомьтесь с [порядок использования современных проверки подлинности (ADAL) с Скайп для бизнеса](https://go.microsoft.com/fwlink/p/?LinkId=785431) , узнайте о том, как работает с Скайп для бизнеса. 
  
Клиенты Office 2016 поддерживают современных проверки подлинности по умолчанию и никаких действий не требуется для клиента для использования этих новых потоков. Тем не менее явно необходим для использования проверки подлинности.
